Hello World – HTML

This is a basic HTML project that displays “Hello, World!” in the browser.
It demonstrates the fundamental structure of an HTML5 webpage.

📂 File Structure
index.html
💻 Code
<!DOCTYPE html>
<html>
<head>
    <title>Hello World</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
▶️ How to Run

Open index.html in any web browser.
